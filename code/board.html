board module

class board.Board(width, height, tile_bag=None, tile_list=None)[source]

Bases: object

Represents the state of the board. This is initialised either by drawing a set of tiles from a Tile Bag and assigning random orientations for by specifying an explicit list of tiles.

width

X-dimension of board in tiles in x direction (west and east)

Type

int

height

Y-dimension of board in tiles y direction (north and south)

Type

int

nattr

Number of board square attributes (NATT), where the attributes are TILE (0) is the tile number ROT (1) is the rotation or orientation of each tile (degrees clockwise)

Type

int

size

Total number of tiles on board (w * h)

Type

int

placements

Holds all information on the state of the each square on the board. Has dimensions h, w, n

Type

Check if exit exists in a particular direction from a tile on the board.

  • dir (int) – Direction in which presence of door to be checked.

Keyword Arguments

bool

slide_line(dir, x_or_y, tile_bag=None)[source]

Slide a row or column

Parameters

  • dir (int) – Direction in which to slide tiles

  • x_or_y (int) – X or y position on board, depending on direction. For dir = NORTH or SOUTH, this is x, indicating the column. For dir = EAST or WEST, this is y, indicating the row.

  • tilebag – Bag of tiles from which new one can be drawn

Return type

Tuple[Rect, ndarray]

Returns

  • Rectangle of tiles to slide, including new tile

  • Holds all information on the state of the each square on the board in the ‘patch’ (this will be either a row or column and one longer than the board, e.g. for a raw (1, w+1, n)
